# Hermes Plans

Use this directory for durable project context that must survive chat compaction.

Recommended files:

- `YYYY-MM-DD-design-spec.md`
- `YYYY-MM-DD-implementation-plan.md`
- `YYYY-MM-DD-validation-log.md`
- `YYYY-MM-DD-retrospective.md`

Rule: if the information affects implementation, review, or acceptance, it belongs here or in `docs/`, not only in chat.
